Method 1: CSV File Import (Recommended)

Import data from the existing USL Google Sheets CSV exports. This is the primary method for migrating from the Google Sheets system.

⚠️ Important Notes

  • This will overwrite existing data in the database
  • Make sure CSV files are in the correct format
  • Run this on a backup/test database first
  • CSV files should be located in /home/tawnyuh/documents/programming/rocket_league/USL/Google_Drive/test_data/
1

Prepare CSV Files

Ensure your CSV files are in the correct directory:

  • User.csv - User data with Discord IDs, names, MMR, TrueSkill values
  • UserTracker.csv - Tracker data with URLs and playlist statistics
2

Run Import Script

Execute the import script from the project directory:

go run internal/usl/scripts/import.go
3

Verify Import

Check the results:

  • Review server logs for any errors
  • Visit Users page to verify user data
  • Visit Trackers page to verify tracker data

Method 3: Advanced Import Options

Advanced tools for specific import scenarios. Use with caution.

1

Database Reset

Custom CSV Format

Edit internal/usl/scripts/import.go to match your CSV structure.

3

Data Validation

Run data validation after import to check for issues:

-- Check for duplicate Discord IDs: SELECT discord_id, COUNT(*) FROM usl_users GROUP BY discord_id HAVING COUNT(*) > 1; -- Check for orphaned trackers: SELECT ut.discord_id FROM usl_user_trackers ut LEFT JOIN usl_users u ON ut.discord_id = u.discord_id WHERE u.discord_id IS NULL;
